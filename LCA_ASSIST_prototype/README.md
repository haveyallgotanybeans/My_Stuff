# Lane Change Assist Prototype (Arduino) 
 
## Overview 
A prototype implementation of a lane change assist system using Arduino. The system uses 5 ultrasonic sensors to detect vehicles in adjacent lanes and autonomously steers to a safe lane when possible. 
 
## Features 
- 5 ultrasonic sensors for front, side, and rear detection 
- Buzzer and LED alerts for warning states 
- Automatic servo-based steering control for lane switching 
- LCD display for sensor feedback 
 
## Hardware 
Arduino board, Ultrasonic sensors (HC-SR04), Servo motor, I2C LCD, Buzzer, LEDs

# Key Localization (MATLAB) 
 
## Overview 
MATLAB-based project for detecting and localizing specific keypoints or reference patterns in images using image processing and correlation methods. 
 
## Features 
- Implements 2D correlation-based localization 
- Supports grayscale and RGB images 
- Visual output with bounding boxes marking localized regions 
- Adjustable matching threshold for accuracy control 
 
## Tools Used 
MATLAB Image Processing Toolbox

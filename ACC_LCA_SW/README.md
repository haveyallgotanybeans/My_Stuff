# Adaptive Cruise and Lane Change Assist (Python) 
 
## Overview 
Python simulation of an adaptive lane change assist system. User provides vehicle speed as input, and the program generates a virtual environment to monitor lane availability, oncoming vehicles, and adjust speed or switch lanes accordingly. 
 
## Features 
- Virtual environment generation with simulated lanes and traffic 
- Dynamic speed adjustment based on vehicle distance and safety margin 
- Automatic lane change when target lane is clear 
- Safety distance threshold of 250ms (configurable) 
 
## Usage 
python acc_lca_sim.py

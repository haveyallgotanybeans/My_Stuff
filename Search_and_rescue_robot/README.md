# Search and Rescue Robot 
 
## Overview 
A robotic system designed for autonomous navigation in rough terrain to locate victims using sensors and camera input. Developed as a proof-of-concept for post-disaster environments. 
 
## Features 
- Ultrasonic and IR sensors for obstacle detection 
- Camera module for basic visual feedback 
- Autonomous movement using motor control logic 
- Buzzer alert when target detected 
 
## Tools and Components 
Arduino, Ultrasonic sensors, Camera module, DC motors, Motor driver
